# Blender_L4D2_Character_Tools
这是一个旨在提升求生之路2人物Mod制作效率的插件。

## 特性

- 运用预设的骨骼映射关系快速进行骨骼对齐与合并
- 插入动作关键帧辅助生成VRD文本
- 快速编写你的抖动骨骼参数
- 通过操控原始形态键的混合数值生成适配你面部规则的表情集
